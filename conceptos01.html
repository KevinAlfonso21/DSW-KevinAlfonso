Etiquetas, atributos y valores

Etiquetas

Imagen recuperada de: https://monroepl.org/wp-content/uploads/2016/12/HTML_logo.png


Etiquetas HTML básicas Hay una serie de etiquetas que son las más usadas para crear cualquier documento HTML, a continuación las explicamos:

body para el contenido

head para información sobre el documento

div división dentro del contenido

a para enlaces

strong para poner el texto en negrita

br para saltos de línea

H1…H6 para títulos dentro del contenido

img para añadir imágenes al documento

ol para listas ordenadas

ul para listas desordenadas

li para elementos dentro de la lista

p para parágrafos

span para estilos de una parte del texto


Para el desarrollo de esta pagina web, se utilizaron la mayoria de las etiquetas anterior mente mencionadas, en general son utilizaras para insertar contenido en nuestro sitio web.

Todos las etiquetas o elementos estan en la siguiente referencia oficial de HTML5 y fue con la que se brindo el concepto en este sitio web. Referencia: MDN web docs. (2020). Lista de Elementos HTML5. de Mozilla. Recuperado de: https://developer.mozilla.org/es/docs/HTML/HTML5/HTML5_lista_elementos

Atributos

Imagen recuperada de: https://misapuntesyacimientos.files.wordpress.com/2016/01/atributos-1.png?w=840


Cada uno de los elementos en HTML5 tiene sus respectivos atributos que son escenciales para la configuración o ajuste del comportamiento de los elementos para cumplir con las necesidades que puedan surgir por los usuarios finales, a continuación se listan los atributos más comunes:

align para especificar el alineamiento horizontal de un elemento

alt el cual sirve para mostrar un texto alterno si una imagen no carga por algún motivo

border el cual especifica el ancho del borde

color el cual es común mente utilizado para establecer el color de texto utilizando un nombre de color o en formato hexadecimal

height el cual especifica los elementos enumerados

href el cual nos sirve para especificar una url de un recurso en internet

label el cual especifica el título de la pista con un formato legible por el usuario

list el cual identifica una serie de valores predefinidos para ser desplegados al usuario final

maxlength el cual es utilizado para definir el mayor numero de caracteres aceptados

name el cual sirve para darle nombre a un elemento para ser identificado como un campo


Todos las atributos estan en la siguiente referencia oficial de HTML5 y fue con la que se brindo el concepto en este sitio web. Referencia: MDN web docs. (2019). Referencia de Atributos HTML. de Mozilla. Recuperado de: https://developer.mozilla.org/es/docs/Web/HTML/Atributos
